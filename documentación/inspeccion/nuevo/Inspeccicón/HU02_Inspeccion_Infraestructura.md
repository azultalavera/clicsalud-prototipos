# HU.02: Auditoría de Infraestructura (Salas y Camas)

**Como** Inspector,
**Quiero** relevar la cantidad física de salas y camas en cada área del establecimiento,
**Para** verificar si lo que se ve en la visita coincide con lo que el efector declaró en los papeles.

### Criterios de Aceptación:

1. **Tabla de Comparación**: Se debe mostrar una tabla que compare lo declarado originalmente en el trámite, las aclaraciones previas del establecimiento y una columna vacía para que el inspector ingrese lo que está viendo realmente.
2. **Ingreso de Valores**: El espacio para que el inspector anote las cantidades debe aceptar únicamente números.
3. **Alerta Visual**: Si la cantidad observada por el inspector es menor a la que el establecimiento había declarado, el sistema debe resaltar ese renglón con un color de advertencia.
4. **Estado de Cumplimiento**: Se debe incluir un botón para que el inspector indique directamente si ese espacio físico "Cumple" o "No Cumple" con lo esperado.
5. **Justificación de Faltas**: Si el inspector marca que "No Cumple", el sistema debe obligarlo a escribir un comentario explicando el motivo del problema.
