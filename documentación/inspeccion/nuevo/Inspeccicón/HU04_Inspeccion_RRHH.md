# HU.04: Validación de Personal y Jefes de Servicio

**Como** Inspector,
**Quiero** constatar la presencia y habilitación de los profesionales declarados,
**Para** asegurar que el establecimiento cuente con el personal exigido por las normativas.

### Criterios de Aceptación:

1. **Listado de Profesionales**: Se debe mostrar la lista de profesionales declarados por el establecimiento para que el inspector confirme quiénes están presentes en ese momento.
2. **Confirmación de Presencia**: Se debe poder marcar con una casilla o botón si cada persona está físicamente en el lugar durante la visita.
3. **Control de Credenciales**: El sistema debe permitir al inspector cruzar el número de matrícula que figura en el sistema con la credencial física que muestra el profesional.
4. **Validación de Jefaturas**: Se debe verificar de forma separada que la persona nombrada como Jefe de Servicio se encuentre formalmente designada para esa tarea.
5. **Anotaciones sobre el Personal**: Debe haber un espacio donde el inspector pueda dejar comentarios sobre cualquier irregularidad relacionada con los profesionales, como ausencias injustificadas.
