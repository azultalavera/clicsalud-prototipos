# HU.05: Gestión de Observaciones y Evidencias

**Como** Inspector,
**Quiero** documentar con lujo de detalles las fallas encontradas durante la visita,
**Para** tener pruebas claras que respalden el resultado del acta y le sirvan al establecimiento para corregir sus errores.

### Criterios de Aceptación:

1. **Relación con la Falla**: Cada observación o comentario negativo debe estar vinculado directamente a un área, equipo o profesional específico.
2. **Captura de Pruebas**: El sistema debe permitir tomar fotografías y asociarlas a la observación, guardándolas de forma segura de inmediato.
3. **Ubicación y Hora**: Las fotos tomadas deben guardar automáticamente la fecha, la hora exacta y el lugar geográfico para tener validez oficial.
4. **Resumen de Problemas**: Antes de terminar la visita, la tablet debe mostrar una pantalla que agrupe y resuma todas las fallas encontradas.
5. **Gravedad del Problema**: El inspector debe poder clasificar cada falla según su importancia (Leve, Grave, Crítica), lo cual influirá en si el establecimiento es aprobado o no.
