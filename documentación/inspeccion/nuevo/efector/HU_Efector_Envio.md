# HU.09: Envío de Rectificación y Notificación

**Como** Responsable de la Clínica (Efector),
**Quiero** enviar mis respuestas y evidencias al inspector,
**Para** que se proceda con la revisión de las correcciones y se avance con el trámite.

### Criterios de Aceptación:

1. **Revisión Final**: Antes de enviar, el sistema debe mostrar un resumen de todos los archivos y descargos cargados.
2. **Control de Obligatoriedad**: No se debe permitir el envío si hay ítems observados que no tengan al menos un documento o descargo adjunto.
3. **Confirmación de Envío**: Al presionar "Enviar", el sistema debe solicitar una confirmación final y advertir que el trámite ya no podrá ser editado.
4. **Flujo de Retorno**: Una vez enviado, el trámite debe regresar automáticamente a la bandeja del inspector original para su re-evaluación.
5. **Comprobante de Presentación**: El sistema debe generar un aviso o comprobante digital confirmando la recepción de la respuesta de emplazamiento.
