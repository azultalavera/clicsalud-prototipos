# HU.08: Carga de Documentación de Subsanación

**Como** Responsable de la Clínica (Efector),
**Quiero** adjuntar pruebas de la corrección de las faltas observadas,
**Para** acreditar ante el Ministerio que la clínica cumple con los requisitos mínimos.

### Criterios de Aceptación:

1. **Vínculo Directo**: Por cada observación del inspector, el efector debe tener un campo dedicado para subir la respuesta.
2. **Formatos de Archivo**: El sistema debe aceptar archivos en formato PDF, JPG y PNG.
3. **Descargo de Texto**: Se debe permitir escribir una nota aclaratoria o descargo por cada ítem rectificado.
4. **Validación de Carga**: El sistema debe indicar visualmente cuáles observaciones ya tienen respuesta cargada y cuáles faltan completar.
5. **Guardado Parcial**: El efector debe poder guardar el progreso de sus respuestas sin necesidad de enviar el trámite completo inmediatamente.
