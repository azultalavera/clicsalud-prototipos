# 🏥 Sistema de Diseño ClicSalud
Bienvenido al manual interactivo del Backoffice de Rugepresa.

---

## 🎨 Identidad Institucional
El color azul es nuestra marca principal. Debe usarse en todos los encabezados y acciones definitivas.

* **Primario:** `#005596` (Azul Institucional)
* **Secundario:** `#29B6F6` (Azul Ciano)
* **Éxito:** `#10B981` (Verde CIDS)

---

## 🔘 Botones de Acción
Los botones deben usar un `borderRadius` de `8px`.

> **Nota para Devs:** Usar `<Button variant="contained">` para acciones primarias.

---

## 📋 Tabla de Equipamientos (Alta Densidad)
Reglas de la tabla de laboratorio:
1. **Navegación Lateral:** Para separar por Servicio.
2. **Inputs:** Usar `InputBase` sin bordes para carga masiva.
3. **Validación:** El borde del input cambia a `#EF4444` si hay error.
